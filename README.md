# PHP Form Sederhana

## Deskripsi
Project ini adalah form sederhana berbasis PHP yang digunakan untuk input data pengguna.

## Fitur
- Input First Name
- Input Last Name
- Input Phone Number
- Input Address
- Menampilkan hasil input setelah submit
- Menggunakan konsep OOP (Class dan Object)

## Struktur File
- `index.php` → halaman utama dan form
- `Person.php` → class Person
- `style.css` → styling halaman

## Cara Menjalankan
1. Simpan project di folder `C:\laragon\www\`
2. Jalankan Laragon
3. Buka browser
4. Akses:
   `http://localhost/php-form-sederhana/`
